### Framework folder structure

testscripts/: 
    All test scripts will be residing in this folder.

uiautomation/: 
    Framework libraries.

uiautomation/pages:
    All page objects.

baseline/:
    Baseline data.

testresult/:
    Test result.

testreport/:
    Test report.

## Running the tests

Please run the test scripts in the folder "testscripts/"
